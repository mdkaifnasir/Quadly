# Campus App

A social media application for campus connectivity, built with CodeIgniter.

## Features
- User Authentication (Login/Register/Forgot Password)
- Feed with Posts and Reposts
- Three-dot "More Options" menu for contextual actions (Save, Mute, Block, Report, etc.)
- Profile Management
- Direct Database Updates via helper scripts

## Setup
1. Clone the repository.
2. Import `database.sql` into your MySQL database (default name: `campus_app`).
3. Update `application/config/database.php` with your database credentials.
4. Set up a local server (e.g., XAMPP) and point it to the project directory.
